## üîß Installation Instructions

1. **Create sounds directory**:
   ```bash
   mkdir -p an_fsq7_simulator/public/sounds
   ```

2. **Download each sound**:
   - Visit each Freesound.org link above
   - Click the download button (may require free account)
   - Save with the filename listed under "Save as"

3. **Convert to web format** (optional but recommended):
   ```bash
   # Convert to OGG for better web compatibility
   ffmpeg -i input.wav -codec:a libvorbis -qscale:a 5 output.ogg
   ```

4. **Verify files**:
   ```bash
   ls -lh an_fsq7_simulator/public/sounds/
   ```

## üéöÔ∏è Audio Processing Tips

For best results, process sounds with Audacity (free):

1. **Normalize** all sounds to -3dB to prevent clipping
2. **Remove silence** from start/end (< 0.1 seconds)
3. **Add subtle reverb** to button clicks (room ambience)
4. **Loop points** for ambient sounds (fade in/out for seamless loops)
5. **Compress dynamics** slightly for consistent volume

Example Audacity chain:
- Effect ‚Üí Normalize (-3dB)
- Effect ‚Üí Noise Reduction (if needed)
- Effect ‚Üí Compressor (Ratio: 3:1, Threshold: -12dB)
- Effect ‚Üí Fade In/Out (for loops)
